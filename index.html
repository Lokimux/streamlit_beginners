streamlit python for beginners

Streamlit is a powerful and easy-to-use Python library that lets you create interactive web applications for machine learning and data science with minimal effort. Whether you're a beginner or an experienced developer, this guide will walk you through the essential functions and methods you need to know to get started with Streamlit.

Getting Started with Streamlit

Installation

pip install streamlit

Running Your First App

streamlit run your_script.py

Importing Streamlit

import streamlit as st

Displaying Text

  • st.title('Title'): Displays a title.
  • st.header('Header'): Displays a header.
  • st.subheader('Subheader'): Displays a subheader.
  • st.text('Text'): Displays plain text.
  • st.markdown('**Markdown**'): Displays text formatted with Markdown.
  • st.code('print("Hello World")'): Displays a code block.
  • st.latex(r'2+2=4'): Displays LaTeX formatted text.

Displaying Data

  • st.write(data): Writes any object (text, data, Markdown, etc.) to the app.
  • st.dataframe(df): Displays a DataFrame as an interactive table.
  • st.table(df): Displays a static table.
  • st.json(json_obj): Displays JSON.

Media Elements

  • st.image(image): Displays an image.
  • st.audio(audio): Displays an audio player.
  • st.video(video): Displays a video player.

Displaying Widgets

  • st.button('Button'): Creates a button.
  • st.checkbox('Checkbox'): Creates a checkbox.
  • st.radio('Radio', options): Creates a set of radio buttons.
  • st.selectbox('Selectbox', options): Creates a dropdown select box.
  • st.multiselect('Multiselect', options): Creates a multiselect.
  • st.slider('Slider', min_value, max_value): Creates a slider.
  • st.select_slider('Select Slider', options): Creates a select slider.
  • st.text_input('Text Input'): Creates a text input box.
  • st.text_area('Text Area'): Creates a text area.
  • st.number_input('Number Input'): Creates a number input.
  • st.date_input('Date Input'): Creates a date input.
  • st.time_input('Time Input'): Creates a time input.
  • st.file_uploader('File Uploader'): Creates a file uploader.
  • st.color_picker('Color Picker'): Creates a color picker.

Control Flow

  • st.stop(): Stops execution and prevents updating the app.

Layouts and Containers

  • st.sidebar: Access sidebar elements.
  • st.beta_expander('Expander'): Creates an expandable container.
  • st.beta_container(): Creates a container for elements.

Status Elements

  • st.spinner('Spinner'): Displays a spinner while executing.
  • st.progress(progress): Displays a progress bar.
  • st.success('Success'): Displays a success message.
  • st.info('Info'): Displays an informational message.
  • st.warning('Warning'): Displays a warning message.
  • st.error('Error'): Displays an error message.
  • st.exception(e): Displays an exception.

Plotting and Charts

  • st.line_chart(data): Displays a line chart.
  • st.area_chart(data): Displays an area chart.
  • st.bar_chart(data): Displays a bar chart.
  • st.pyplot(fig): Displays a Matplotlib figure.
  • st.altair_chart(chart): Displays an Altair chart.
  • st.vega_lite_chart(data, spec): Displays a Vega-Lite chart.
  • st.plotly_chart(fig): Displays a Plotly chart.
  • st.bokeh_chart(fig): Displays a Bokeh chart.
  • st.deck_gl_chart(pydeck_obj): Displays a PyDeck chart.
  • st.graphviz_chart(dot): Displays a Graphviz chart.

Utilities

  • st.cache(func): Caches a function to avoid re-running it.
  • st.set_page_config(title, layout, initial_sidebar_state): Sets the page configuration.

Example App


import streamlit as st
import pandas as pd
import numpy as np

# Title and text
st.title('My First Streamlit App')
st.text('This is some text.')

# Display a dataframe
df = pd.DataFrame(np.random.randn(10, 5), columns=('col %d' % i for i in range(5)))
st.dataframe(df)

# Interactive widgets
if st.button('Say hello'):
    st.write('Why hello there!')
else:
    st.write('Goodbye')

# Slider
value = st.slider('Select a value', 0, 100)
st.write('Value selected:', value)

# Line chart
chart_data = pd.DataFrame(np.random.randn(20, 3), columns=['a', 'b', 'c'])
st.line_chart(chart_data)

# Sidebar
st.sidebar.header('Sidebar header')
sidebar_value = st.sidebar.slider('Select a sidebar value', 0, 100)
st.sidebar.write('Sidebar value selected:', sidebar_value)
